VO Preparer

Purpose

  • Turn film scripts from excel into word format
  • Automatically filters out captions

How To

  • Open the program
  • Select all the excel files you want to convert
  • The files will be saved in a subfolder called "docs"

Remarks

  • The caption filtering only works if the captions have the correctly formatted gray color
  • Due to this, please make sure that the captions are not marked in other colors than gray
  • For performance reasons, the filter only works for captions with maximum 6 lines per timecode – should there be more necessary, please contact me to adapt the script
