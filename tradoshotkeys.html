Trados Hotkeys

Purpose

  • A collection of several scripts to make working with Trados a little bit easire

How To

  • Alt + Enter / Alt + NumpadEnter: The English segment in Trados will be copied and filtered for in the most recently opened PDF - that way you can immediately jump in the reference file to the segment you are QCing
    • Obviously, the script has its limitations: if the text in the PDF for example has a hard return and the segment in Trados doesn't, it wouldn't be found
  • Alt + Numpad+ - Filter for the complete source segment
  • F9 - Insert the English in the segment, approve segment and move to next one
  • Numpad0 + Numpad2 - Activate the Trados window
  • Ctrl + Alt + F12: Set up the Trados tab jumper (see below)

Trados Tab Jumper

Purpose

  • Quickly jump between several tabs in Trados
  • This also applies the most recent filter which makes it especially useful in CC1

How To

  • Hold Numpad2 and then press any number on the number strap (above the keys) to jump to this tab
    • 1 jumps to tab 1, 2 jumps to tab 2 etc.

How to start

  • Press Ctrl + Alt + F12
  • After the message box has been popped up, activate your Trados and press "Scroll Lock" (upper right on your keyboard)
  • This will set the window size of Trados as this is needed for the script (if you need to resize Trados, please relaunch the application)
