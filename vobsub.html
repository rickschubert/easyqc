VobSub Exporter

Purpose

  • For Final QC
  • Automatically export Subtitle Edit ready text files (film scripts; optimally, prepared beforehand with the easyQC Subtitle Converter) via Subtitle Edit into VobSub files so that you can overlay the subtitle in VLC

How To

  • Simply open the application and follow the instructions
  • The big plus: you don’t even need to have Subtitle Edit anymore on your PC, as this application will install it automatically for you with all necessary settings

Remarks

I tested this script on 4 different Windows 7 PCs together with text files for the project CIM1 Refresh Russian. Should it not work with some projects, please don’t hesitate to send me an e-mail: it may be possible that the original scripts are in a bad format (already happened with NCRC Swedish) or that the scripts are maybe a lot longer than I expect. In the latter case, it should be easily possible to amend the wait times of my application so that it can stomach harder files more easily.
