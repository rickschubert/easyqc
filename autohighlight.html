AutoHighlight PDFs

Find and Mark Batch Processing (Actions) script by Thom Parker, WindJack Solutions, Inc. October 2010
www.windjack.com, www.pdfscripting.com
For Adobe Systems Inc., www.adobe.com

This script has been done by Thom Parker. The following document is merely a guideline on how to use Parker's phenomenal work.

My only work regarding this is the script to ease up compiling glossaries.

Installing the script

  1. Get the “AutoHighlight.sequ” from the QA Team references folder
  2. Double click it – it should now be part of your Acrobat Reader

Setting up the script to mark up files

  1. Create a text file (Unicode) of the text entries you want to highlight
    • Create an excel list with terms – to make it easier, we have a set of AutoHotkeys available (see AutoHotkeys to compile glossaries)
    • Save the excel list with Unicode: “File → Save As → Unicode Text”
      • Important: Create a list like this for each language you want to highlight – thus, only have the English in the excel file first and save it as Unicode and then only the Translation terms which you save as Unicode
  2. In the top right “Tools | Sign | Comment”: click “Tool”
    • Locate “Action Wizard”
    • Right click on “AutoHighlight” → “Edit Action”
  3. In the right window, click “Search & Remove Text”
    • Click “Specify Settings”
    • Click “Import” and locate the text files of files you want to highlight
    • o Click “Okay”
  4. Click “Save” and again “Save”

Notes before running the script

  • Please copy the files to mark up on your PC as the script is set up that way that the PDF files will be automatically saved with the mark-ups
  • The dialogue with the folder to locate with files to mark up does not allow copying in links: We suggest having the files in a folder on your desktop so that you will find them easily

Running the script

  1. Select “Tools” and click then on “AutoHighlight”
    • Tip: If you right click on “AutoHighlight” and select “Add to Quick Tools Toolbar”, you will have the script always ready at the top bar
  2. Click “Add folder”
    • Add the folder with files you want to mark up
    • Click “Start”

Remarks

  • This has been tested on Adobe Acrobat XI Pro – if it is not working due to another version, just let us know, we are sure that it also works with other versions of Adobe PDF (at least with Acrobat X and higher), there might only be a slight change in how to install it
  • This tool uses the PDF search function: Thus, if a word appears in another word, the whole word will highlighted
    • Example
    • The glossary file has ir
    • The PDF has the word Irokese
    • ↷ the whole word Irokese will be marked up
    • Due to that, it might be better to avoid looking for such small words as the number of false hits might get very high

Compiling glossaries AutoHotkey Script

  • The scripts can be found in the QA Team folder (the one with the chili)
  • Ctrl + Shift + X - Copy any highlighted text into the most recent excel file
    • The text will be copied into the excel file, the cursor will jump one cell below and the window which has been active before will be activated again
  • Ctrl + Shift + Z - As the previous feature, but in zigzag mode jumping between two excel columns
  • Pause - Reset the zigzag mode (Ctrl + Shift + Z) to the left column
  • ScrollLock - Toggle the Clipboard Cleaner on/off
    • The clipboard cleaner automatically removes all formatting as well as returns and double spaces from selected text. This mode is by standard turned on, can be turned off though should the language make it necessary (i.e. languages without word spaces).
  • Alt + A - Amend hard returns
    • Position the cursor on the end of a line; the next line will be brought up and a space inserted
  • Numpad3 & H - Delete checkpoints/the last word from cells in an excel file
  • Numpad3 & Y - Delete first word/checkpoint from excel cells (i.e.EXW - Ex-Works)
