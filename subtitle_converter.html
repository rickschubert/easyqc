Subtitle Converter

Purpose

  • For Final QC
  • Convert film scripts (Excel format) into tab-delimited text files so that overlays can be easily done with the tool Subtitle Edit
  • Captions will automatically be filtered out as well

How To

  • Open the file
  • Select all the excel files you want to convert and that's it: the files will be implemented in the same folder the excel files are

Remarks

  • The script will also export the reduced/filtered excel files so that you can check for yourself that everyting is all right
  • All Excel and Word applications should be closed before starting this application (The script will tell you if you these programs are still open)
