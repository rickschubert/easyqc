Script Preparer for AutoHighlightPDF Final QC

Purpose

  • Auto-prepares Film Scripts (Excel) for the AutoHighlight PDF tool (Final QC)

How To

  • Double click on the software
  • Select the files you want to turn into PDFs and click “Open”
  • The program creates a subfolder called “PDFs” in the same location the film scripts lie in and saves there the new PDF files
