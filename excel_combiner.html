Excel Combiner

Purpose

  • To combine several Excel files (film scripts) into one big master workbook
  • This makes global searches very easy (see below)

How To

  • Start the application
  • Select all the files (with Shift on your keyboard) you want to merge into one file
  • That's it!

Remarks

  • The script also renames the tabs of the worksheets (see below)
    • For this purpose, the script takes the last 31 characters of the file names (as these are usually more unique than the beginning ones)
  • Big thanks to the user "kon" from the AutoHotkey community who helped me with this one!
